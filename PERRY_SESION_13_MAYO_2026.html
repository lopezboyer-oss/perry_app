Decisión Arquitectónica Clave

🏗️ Excel Multiempresa: API Route Dedicada

La funcionalidad Excel Multiempresa se implementó como un API Route separado (/api/weekend-export-all) en vez de extender la lógica del cliente. Esto permite query directo a la BD sin limitaciones de cookie de empresa activa, genera el archivo en el servidor con datos de todas las empresas, y mantiene el control de acceso centralizado (solo ADMIN + SAFETY_LP). La columna "Empresa" se agrega automáticamente al inicio para identificar cada actividad.
